# The Zen Den | Yoga Studio Website

A minimalist, high-performance yoga studio website built with Next.js 14, Tailwind CSS, and Framer Motion. This project demonstrates a "Japanese-Scandi" aesthetic with advanced responsive layouts and smooth animations.

The Zen Den is a portfolio piece designed to showcase:
- **Complex CSS Grid Layouts**: A responsive timetable that transforms from a grid on desktop to a stacked list on mobile without using tables.
- **Design System Implementation**: A cohesive visual language using custom Tailwind configuration for colors (Sage, Alabaster, Terracotta) and typography (Tenor Sans, Lato).
- **Advanced Animations**: Soft-load page transitions and scroll-triggered effects using Framer Motion.
- **Mobile-First UX**: Features like a sticky "Book Now" button that only appears on mobile devices to improve conversion.
